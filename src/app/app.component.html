Pipes in Angular

Pipes are used to transform and manipulate data on the screen.

Examples of Built-in Pipes

  • DatePipe:

    Original: {{ today }}

    Formatted:

    | date: {{ today | date }}

    | date:'short': {{ today | date:'short' }}

    | date:'fullDate': {{ today | date:'fullDate' }}

    Custom Formatted:

    | date:'fullDate': {{ today | date:'dd/mm/yyyy HH:mm:ss' }}

  • LowerCasePipe:

    Original: {{ 'Hello Angular' }}

    Lowercase: {{ 'Hello Angular' | lowercase }}

  • UpperCasePipe:

    Original: {{ 'Hello Angular' }}

    Uppercase: {{ 'Hello Angular' | uppercase }}

  • CurrencyPipe:

    Original: {{ price }}

    Formatted: {{ price | currency:'INR' }}

    Formatted: {{ price | currency:'USD':'symbol' }}

  • SlicePipe:

    Original: {{ 'Hello Angular Pipes' }}

    Sliced: {{ 'Hello Angular Pipes' | slice:6:13 }}

  • Add Number and JSON Object

    Number: {{ 5 }}

    Object: {{ user1 | json }}
